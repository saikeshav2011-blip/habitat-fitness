# Habitᵃᵗ Fitness

## Goal

Habitᵃᵗ Fitness is an interval workout app designed to act as a personal fitness coach.

Users can create workouts, discover exercises, learn proper movement techniques, track progress, and receive workout guidance even when they leave the app.

The goal is to make workouts simple to follow while helping users build consistent fitness habits.

---

# Core Features

## 1. Workout Creation

Users can create custom workouts.

Features:

- Create workout names
- Add exercises
- Set work duration
- Set rest duration
- Arrange exercise order
- Edit workouts
- Save workouts
- Restart previous workouts

Users can choose exercises from:
- Public Exercise Library
- Their own private exercises

Example:

Morning HIIT

1. Jumping Jacks
   - Work: 30 seconds
   - Rest: 15 seconds

2. Squats
   - Work: 45 seconds
   - Rest: 15 seconds

# 2. Exercise Library

The Exercise Library teaches users how exercises work.

## Public Exercises

Public exercises are available to all users.

Each exercise may include:

- Exercise name

- Target muscle groups

- Chest
- Shoulders
- Triceps
- Core

- No Equipment

# 4. Exercise Submission System

Users can optionally submit private exercises to become public exercises.

Flow:

User creates exercise

↓

Exercise remains private

↓

User chooses "Submit to Habitᵃᵗ Fitness"

↓

Admin reviews submission

↓

Admin can:
- Approve
- Edit
- Add information
- Reject

Approved exercises become part of the public library.

# Home Workout Philosophy

Habitᵃᵗ Fitness is designed primarily for home workouts.

The app should prioritize exercises that require:

- No equipment
- Minimal equipment
- Affordable home equipment

Supported equipment examples:

- None
- Yoga mat
- Resistance bands
- Dumbbells
- Jump rope
- Chair
- Pull-up bar

The app should avoid recommending exercises that require:

- Commercial gym machines
- Expensive specialized equipment
- Hard-to-access facilities

Workout generation should allow users to select available equipment and only recommend appropriate exercises.

# Multi-Platform Support

Habitᵃᵗ Fitness will support both mobile and web platforms.

Mobile focus:
- Workout execution
- Timers
- Notifications
- Lock screen workout guidance

Web focus:
- Workout planning
- Exercise browsing
- Workout management
- Progress review

The app should use a shared codebase whenever possible.
