# Habitᵃᵗ Fitness

## Goal
A fitness interval timer app that guides users through workouts with exercise animations, timers, and notifications.

## Core Features

- Interval workout timer
- Exercise list with numbers and names
- Cartoon exercise illustrations
- Rest/work intervals
- Notifications when user leaves the app
- Workout history
- Custom workouts

## Platforms

- Mobile (iOS + Android)
- Web
